Talking Points
Start with their world, not yours. “I wanted to start by understanding what’s changed for your team since earlier this year and what your top priorities look like right now.”
Anchor on outcomes, not features. Reference the value delivered before the champion left — usage data, security incidents prevented, time saved on access management.
Propose the extension as a win for them. “I want to make sure you have enough time to feel confident in this decision. Would a 60-day extension give you what you need?”
Objection Handlers
“I’m not sure we’re getting value from this.”
Pull up the usage report in the call. Show active users, vaults accessed, and security events this quarter. Make the value visible before responding.
“We’re re-evaluating our security stack.”
Validate the review, don’t resist it. Ask: “What criteria are most important to your team?” Then map those criteria to your strengths — don’t list features unprompted.
“The previous team was handling this.”
Acknowledge the transition directly. “That’s exactly why I wanted to reach out — I want to make sure you have full context so you can make the right call for your team.”
